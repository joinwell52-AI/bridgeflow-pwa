# FCoP — 自媒体团队 / Content Media Team

> **主控角色（leader）：PUBLISHER** | 各模板对比：dev-team=PM / mvp-team=MARKETER / media-team=PUBLISHER / qa-team=LEAD-QA

## 团队构成 / Roles

| 角色 | 代码 | 职责 |
|------|------|------|
| 审核发行 | PUBLISHER | 终审、发布排期、任务分发 |
| 素材采集 | COLLECTOR | 热点追踪、素材搜集 |
| 拟题提纲 | WRITER | 选题策划、内容撰写 |
| 润色编辑 | EDITOR | 文字润色、排版优化 |

## 内容流水线 / Content Pipeline

```
PUBLISHER 下达选题 → COLLECTOR 采集素材 → WRITER 撰写初稿 → EDITOR 润色 → PUBLISHER 终审发布
```

> 本模板遵循 FCoP Rule 4（角色链路）：`ADMIN ↔ PUBLISHER` 是唯一对外接口；其他角色只从 `PUBLISHER` 接收任务、只向 `PUBLISHER` 回执。

## 快速开始 / Quick Start

1. 开 4 个 Cursor Agent 窗口
2. 每个窗口分配一个角色
3. 对 PUBLISHER 说"开始工作"
4. 下达第一个选题任务
