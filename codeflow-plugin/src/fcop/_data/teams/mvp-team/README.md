# FCoP — 创业 MVP 团队 / Startup MVP Team

> **主控角色（leader）：MARKETER** | 各模板对比：dev-team=PM / mvp-team=MARKETER / media-team=PUBLISHER / qa-team=LEAD-QA

## 团队构成 / Roles

| 角色 | 代码 | 职责 |
|------|------|------|
| 增长运营 | MARKETER | 推广策略、Landing Page、任务协调 |
| 市场调研 | RESEARCHER | 竞品分析、用户痛点 |
| 产品设计 | DESIGNER | PRD 撰写、交互设计 |
| 快速原型 | BUILDER | 技术选型、MVP 搭建 |

## MVP 流程 / MVP Flow

```
MARKETER 定方向 → RESEARCHER 调研 → DESIGNER 设计 → BUILDER 实现 → MARKETER 推广
```

> 本模板遵循 FCoP Rule 4（角色链路）：`ADMIN ↔ MARKETER` 是唯一对外接口；其他角色只从 `MARKETER` 接收任务、只向 `MARKETER` 回执。

## 快速开始 / Quick Start

1. 开 4 个 Cursor Agent 窗口
2. 每个窗口分配一个角色
3. 对 MARKETER 说"开始工作"
4. 下达第一个调研任务
