## 失败处理

| 错误 | 处理 |
|---|---|
| `npm test` 任一 fail | 报告 PM；不 commit |
| `tsc --noEmit` 失败 | 同上 |
| commit 被 pre-commit hook 拦 | 看 hook 输出，**不**用 `--no-verify`；报告 PM |
| origin `non-fast-forward` | 停手报告 PM（不擅自 rebase） |
| backup 鉴权失败 | 重试 1 次；仍失败 → 报告（origin 成功 = 不阻塞总任务） |
