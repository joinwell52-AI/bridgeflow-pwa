## §7 失败回滚

如任一 HARD GATE 失败或 commit 损坏：

```bash
git reset --soft HEAD~1   # 保留改动，撤销 commit
# 或如已 push：
git push origin :refs/tags/<不存在>  # 不适用，因为不打 tag
git revert HEAD --no-edit && git push origin main backup main  # 不推荐，制造历史污染
```

**首选**：commit 前在 stage 阶段就 abort（`git reset` 清 stage），重新评估，避免 push 出去再 revert。
