# GitHub SEO 优化 — API 命令

需要 GitHub Personal Access Token 执行。也可以在 GitHub 网页 Settings 里手动设置。

---

## 1. joinwell52 仓库 — 补 description + homepage

**网页操作路径:** https://github.com/joinwell52-AI/joinwell52 → Settings → General

- **Description:** `Build a 4-role AI team in Cursor IDE — PM+DEV+OPS+QA collaborate autonomously. 87 person-days in 17 days. Now a real product → CodeFlow`
- **Website:** `https://joinwell52-ai.github.io/joinwell52/`
- **Topics (已设好):** ai-agent, ai-collaboration, ai-development, ai-team, automation, cursor, cursor-ide, developer-tools, multi-agent, productivity

---

## 2. codeflow-pwa 仓库 — 补 description + topics

**网页操作路径:** https://github.com/joinwell52-AI/codeflow-pwa → Settings → General

- **Description:** `CodeFlow (码流) — AI-powered human-AI collaboration hub. Desktop EXE + Mobile PWA + MCP Plugin. Command your AI team from phone.`
- **Website:** `https://joinwell52-ai.github.io/codeflow-pwa/`
- **Topics (新增):** `codeflow`, `cursor`, `ai-agent`, `multi-agent`, `pwa`, `desktop-app`, `mcp`, `human-ai-collaboration`, `ai-team`, `developer-tools`, `cursor-ide`, `open-source`

---

## 3. 开启 Discussions

两个仓库都建议开启 Discussions（Settings → Features → Discussions ✓），方便社区互动。

---

## 4. 仓库 About 设置（右侧栏）

在仓库主页右侧 About 区域（齿轮图标）：
- 勾选 "Releases"
- 勾选 "Packages" (如适用)
- 确保 Website 链接已设置
